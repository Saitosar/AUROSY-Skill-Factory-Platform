# Cyclone DDS (macOS arm64)

Сюда при необходимости кладутся локальные `libddsc.dylib` / `libddscxx.dylib`, если вы собирали Cyclone DDS для разработки на Apple Silicon и не хотите ставить их в `/usr/local`.

Файлы `*.dylib` **не коммитятся** в git по политике репозитория (см. `.gitignore` в этом каталоге). Скопируйте бинарники из своего билда или используйте установку через CMake / пакетный менеджер.
