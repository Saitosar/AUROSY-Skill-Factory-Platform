# Бэкенд: бэклог под персистенцию авторинга (действие / навык / сценарий)

Фронтенд уже поддерживает экспорт/импорт `scenario.json` v1 и перенос keyframes из Студии движений без обязательного хранения на сервере. Ниже — задачи для репозитория **FastAPI**, когда понадобится полная интеграция.

1. **Сценарии на платформе** — `POST`/`GET` (или `POST /api/platform/artifacts/{name}` с соглашением имён `scenario-*.json`) для списка и версий сценариев пользователя.
2. **Регистрация mid-level действия** — загрузка ZIP или серверная генерация папки `basic_actions|complex_actions/<name>/` (эквивалент связки Pose Studio + `action_exporter`), плюс обновление индекса для `GET /api/mid-level/actions`.
3. **Связка Skill Bundle ↔ `library/skills/`** — опциональная выгрузка/импорт в формат папки SDK или документированный маппинг полей.
4. **Валидация сценария на сервере** — проверка нод против фактического каталога действий перед постановкой «обучения сценария» (если появится такой job).

Документ можно уточнять по мере появления маршрутов в OpenAPI бэкенда.
