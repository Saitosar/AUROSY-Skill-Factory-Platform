# F1 — каркас приложения (AUROSY Skill Factory Web UI)

> **Для агентов:** реализация по шагам — superpowers:subagent-driven-development или superpowers:executing-plans. Шаги с чекбоксами `- [ ]`.

**Цель:** стабильный shell (layout + маршруты), единый HTTP-клиент с настраиваемым базовым URL, глобальная индикация недоступности бэкенда по `GET /api/health`, дизайн-токены из [02_design_system.md](../../g1-control-ui/02_design_system.md), заглушка маршрута под будущий Pose-экран (F4).

**Архитектура:** базовый URL API берётся из `import.meta.env.VITE_API_BASE` с fallback на `""` (относительные пути `/api/*` в dev через прокси Vite). Состояние «бэкенд недоступен» поднимается на уровне корневого layout (контекст или локальный стор в `App`): периодический или при монтировании `getHealth()`; при ошибке сети/не-OK — баннер поверх main. Стили: CSS variables в `:root`, существующие классы `.layout` / `.sidebar` переводятся на токены без смены структуры DOM.

**Стек:** React 18, React Router 6, Vite 5, TypeScript. Целевой код: этот репозиторий (**AUROSY_creators_factory**), каталог `web/frontend/` (см. [README.md](../../g1-control-ui/README.md)).

**Исходная спецификация:** [03_implementation_roadmap_frontend.md](../../g1-control-ui/03_implementation_roadmap_frontend.md) — секция F1.

```typescript
const API_BASE = import.meta.env.VITE_API_BASE ?? "";

function apiUrl(path: string): string {
  const p = path.startsWith("/") ? path : `/${path}`;
  return `${API_BASE}${p}`;
}

export async function getHealth(): Promise<{ status: string }> {
  const r = await fetch(apiUrl("/api/health"));
  if (!r.ok) throw new Error(await r.text());
  return r.json() as Promise<{ status: string }>;
}
```

- [ ] **Шаг 1:** Реализовать контекст (интервал опроса 15–30 с, очистка interval при unmount; первая проверка сразу):

```tsx
// BackendStatus.tsx — суть контракта (реализацию дописать полностью в файле)
import { createContext, useContext, useEffect, useMemo, useState, type ReactNode } from "react";
import { getHealth } from "../api/client";

type Status = "unknown" | "ok" | "down";

type Ctx = { status: Status; lastError: string | null; recheck: () => void };

const BackendStatusContext = createContext<Ctx | null>(null);

export function BackendStatusProvider({ children }: { children: ReactNode }) {
  const [status, setStatus] = useState<Status>("unknown");
  const [lastError, setLastError] = useState<string | null>(null);

  const check = () => {
    getHealth()
      .then(() => {
        setStatus("ok");
        setLastError(null);
      })
      .catch((e: unknown) => {
        setStatus("down");
        setLastError(e instanceof Error ? e.message : String(e));
      });
  };

  useEffect(() => {
    check();
    const id = window.setInterval(check, 20000);
    return () => window.clearInterval(id);
  }, []);

  const value = useMemo(() => ({ status, lastError, recheck: check }), [status, lastError]);

  return <BackendStatusContext.Provider value={value}>{children}</BackendStatusContext.Provider>;
}

export function useBackendStatus(): Ctx {
  const v = useContext(BackendStatusContext);
  if (!v) throw new Error("useBackendStatus must be used within BackendStatusProvider");
  return v;
}
```
